# Mobile_Phone_Reference_Application

## 📌 Project Overview

The Mobile Phone Reference Application is a menu-driven application developed in C that provides users with detailed information about various mobile phones. The application serves as a centralized platform for storing, managing, searching, and comparing mobile phone specifications. It enables users to quickly access information such as brand details, processor specifications, RAM and storage capacity, camera features, battery capacity, display size, operating system, and pricing information.

The system includes both User and Data Management modules. Users can browse available mobile phones, search for devices by brand or model name, view complete specifications, compare two phones side by side, and access pricing details. The Data Management module allows administrators to add, update, delete, and display mobile phone records efficiently.

The application utilizes file handling techniques to store phone information in a binary file, ensuring data persistence between program executions. If no existing data file is found, the system automatically loads sample mobile phone records to provide users with initial data for reference.

## 🚀 Key Features

### Data Management
